Consultar CEP

Descrição

Plugin jQuery para preencher campos de formulários a partir de CEP digitado por usuário.

Powered by postmon.com.br


Download

Instalação

Adicione o arquivo minificado jquery.consultar-cep.min.js após o arquivo do jQuery. 

<script src="http://code.jquery.com/jquery-1.10.1.min.js"></script>
<script src="jquery.consultar-cep.min.js"></script>
Código JavaScript
  $('#cep').consultarCep({
    evento:        'blur',
    focarAposPara: '#numero',
    campos: {
      logradouro:  '#endereco',
      bairro:      '#bairro',
      localidade:  '#cidade',
      uf:          '#uf'
    }
  });
