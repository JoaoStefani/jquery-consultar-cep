Exemplo Plugin jQuery Consultar CEP
Consultar CEP ao clicar em botão

Exemplo de código JavaScript
        $('#cep').consultarCep({
          focarAposPara: '#numero',
          mensagem: '<i class="fa fa-spin fa-spinner"></i>',
          classMensagem: 'label label-default',
          btnConsultar:  '#consultar-cep',
          campos: {
            logradouro: '#endereco',
            bairro:     '#bairro',
            localidade: '#cidade',
            uf:         '#uf'
          }
        });
